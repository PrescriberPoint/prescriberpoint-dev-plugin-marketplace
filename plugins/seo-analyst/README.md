# SEO Analyst Plugin

SEO analysis, keyword research, and optimization recommendations with Google Search Console integration.

## Installation

```
/plugin install seo-analyst@prescriberpoint-plugins
```

## Usage

```
/seo <request>
```

## Examples

### Keyword Research
```
/seo What keywords should we target for medication adherence content?
/seo Find quick wins in our GSC data
/seo Research keywords for prescription savings
```

### Technical SEO
```
/seo Check if this URL is indexed: https://example.com/page
/seo Review our sitemap status
/seo What's our search performance for "drug prices"?
```

### Optimization
```
/seo Suggest a better title tag for our homepage targeting "prescription costs"
/seo What schema markup should we add to our medication pages?
/seo How can we improve CTR for queries where we rank position 5-10?
```

## Google Search Console Integration

When GSC is connected, the skill can pull real data:

| Tool | Data |
|------|------|
| `search_analytics` | Impressions, clicks, CTR, average position |
| `detect_quick_wins` | Low-hanging optimization opportunities |
| `index_inspect` | URL indexation status |
| `list_sitemaps` | Sitemap health and coverage |
